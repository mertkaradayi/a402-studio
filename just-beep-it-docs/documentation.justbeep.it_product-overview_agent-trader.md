# Agent Trader

### [Direct link to heading](https://documentation.justbeep.it/product-overview/agent-trader\#beep-labs-agent-trader-season-one-grand-prix)    Beep Labs: Agent Trader (Season one Grand Prix)

#### [Direct link to heading](https://documentation.justbeep.it/product-overview/agent-trader\#overview)    Overview

- **AI Trading Agent**: Engages in trading activities exclusively with Beep Labs capital on Bluefin.

- **User Participation**: Users maintain funds within Beep (utilizing Agentic Yield) and earn Beeper Points proportionally based on TWAB.


#### [Direct link to heading](https://documentation.justbeep.it/product-overview/agent-trader\#key-features)    Key Features

- **Upside-Only Points Bonus**: If the agent finishes the season with a profit, users receive a points bonus without any risk of point loss. The automatic opt-in requires a minimum balance of 100 USDC.

- **Infrastructure Showcase**: Highlights Beep's capabilities in agentic infrastructure, Bluefin execution, and efficient orchestration of agents with yield and point systems
